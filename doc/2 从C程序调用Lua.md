# 从C程序调用Lua
从正在运行的C程序中，你可以调用Lua脚本。C程序可以将参数传递给Lua脚本，而Lua脚本可以向C程序传回一个返回值。Lua脚本可以用来初始化C程序，或者Lua脚本可以在C程序注册一个回调函数。

## 剖析一个C程序对Lua的调用
从C到Lua脚本的典型交互流程看起来像这样：
* 创建一个Lua状态变量
* 加载Lua库
* 加载Lua脚本
* 启动脚本运行以创建脚本的全局变量
* 运行加载的lua脚本
* 从Lua脚本中检索返回
* 关闭Lua状态变量

## 实现一个调用Lua脚本的C程序
实现一个C程序调用Lua脚本的实例，需要实现一下功能：
* 加载的Lua脚本启动运行
* 将参数传递给Lua脚本
* 从Lua脚本中返回
